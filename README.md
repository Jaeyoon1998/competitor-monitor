# 경쟁사 이벤트 · 프로모션 데일리 모니터

경쟁사의 이벤트와 프로모션을 매일 자동 수집하고, 타임라인 대시보드로 보여준다.

```
config/competitors.json   감시 대상 정의 (여기만 채우면 시작됨)
collect.js                수집기 → data/raw/YYYY-MM-DD.json
data/events.json          구조화된 이벤트 저장소 (사실상의 DB)
data/seen.json            중복 방지 지문
build-dashboard.js        events.json → dashboard.html
src/template.html         대시보드 화면
DAILY.md                  매일 실행되는 에이전트가 따르는 절차
```

외부 패키지 의존성 없음. Node 18 이상이면 `npm install` 없이 바로 돈다.

## 시작하기

1. `config/competitors.json` 의 `example-competitor` 를 지우고 실제 경쟁사를 넣는다.
   파일 안 `_fields` 에 각 항목의 의미가 적혀 있다. 모르는 값은 `null` 로 두면 건너뛴다.
2. 수집 → 빌드:
   ```
   node collect.js
   node build-dashboard.js
   ```
3. `dashboard.html` 을 브라우저로 열어 확인한다.

`node collect.js --all` 은 중복 방지를 무시하고 전량 재수집한다 (첫 세팅 때 유용).
`node collect.js --only=<id>` 는 한 경쟁사만 수집한다.

## 수집 대상

| 소스 | 무엇을 얻나 | 자동화 |
|---|---|---|
| App Store 릴리스 노트 | 업데이트에 실린 이벤트·콜라보 공지 | ✅ 공개 API (키 불필요) |
| App Store 인앱 상품 가격 | 재화 팩 구성과 **가격**. 할인·프로모션 감지의 핵심 | ⚠️ 앱 상세 페이지 파싱 (lookup API는 인앱 상품을 주지 않음) |
| Google Play 릴리스 노트 | 안드로이드 쪽 공지 (iOS와 시점이 다를 수 있음) | ⚠️ 페이지 파싱 — 구글이 구조를 바꾸면 깨짐 |
| 공식 유튜브 | 이벤트 예고·보상 영상. 정보량이 가장 많음 | ✅ RSS |
| RSS / Atom 피드 | 블로그, Steam 뉴스 등 | ✅ |
| 공지 게시판 | 공식 홈페이지 공지 목록 | ⚠️ 사이트별 `notice_link_pattern` 조정 필요 |
| Meta / TikTok 광고 라이브러리 | 프로모션 크리에이티브·문구 | ❌ 공개 API 없음 — 주 1회 수동 |
| 인게임 배너 | 가장 정확한 원본 | ❌ 수동 |

마지막 두 줄은 **자동화가 불가능하다**. 광고 라이브러리 API는 정치·이슈 광고만 공개돼
있어 게임 광고는 웹 UI로만 볼 수 있다. `ad_library_url` 필드에 검색 URL을 저장해 두고
주 1회 사람이 확인하는 것을 전제로 설계했다.

## 이벤트 데이터를 채우는 방법

`collect.js` 는 **원자료만** 모은다. 원자료를 이벤트로 해석해 `data/events.json` 에
넣는 일은 Claude가 한다 — 절차는 `DAILY.md` 에 있다. 손으로 넣어도 된다.

`build-dashboard.js` 는 빌드 전에 events.json 을 검증한다 (필수 필드, 유형, 중복 id,
날짜 역전, 미등록 competitor_id). 실패하면 빌드가 중단된다.

## 대시보드

- **타임라인**: 경쟁사별로 묶인 간트. 막대에 마우스를 올리면 요약·키워드·신뢰도가 뜬다.
  종료일이 없는 이벤트는 오른쪽 끝이 페이드된다.
- **표**: 같은 데이터의 전체 목록. `#table` 로 바로 열 수 있다.
- **최근 감지**: 최신 12건과 출처 링크.
- 유형 칩은 범례이자 필터다. 상태(진행 중/예정/종료)는 저장하지 않고 **여는 시점 날짜로
  계산**하므로, 다시 빌드하지 않아도 날짜가 지나면 알아서 종료로 바뀐다.

색상은 색각 이상 검증을 통과한 5색 팔레트를 쓴다. 라이트 모드에서 일부 색이 배경 대비
3:1 미만이라 모든 막대에 텍스트 라벨과 표 뷰를 함께 제공한다 (색만으로 구분하지 않는다).

## 자동 실행

`DAILY.md` 를 따르는 스케줄 에이전트를 매일 아침 돌린다. 등록은 `/schedule` 로 한다.
